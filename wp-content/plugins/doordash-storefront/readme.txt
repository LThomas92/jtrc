Add commission-free online ordering for pickup and delivery to your own website.


== Description ==

Storefront by DoorDash is an online ordering system that allows businesses to incorporate DoorDash ordering, delivery, and pickup logistics directly into their website, with zero commissions. Storefront enables businesses to turn a standard informational site into an e-commerce site, allowing them to own more parts of the customer relationship, including customer insights and ordering data.

Storefront is now available to all restaurants in the U.S., whether or not they are already partnered with DoorDash. You can get started by going to [Storefront website](https://get.doordash.com/products/storefront) or using your DoorDash merchant portal to signup for Storefront.

This creates a unique ordering link for your business. The next step is to add this link to your website. We help you do this using the Storefront Wordpress plugin. The plugin helps you create a custom high visibility ordering button that significantly improves customers ability to place an order on your website. Once you have signed up for Storefront, all you need is your Store ID and Business ID to get the plugin setup. These are available in the DoorDash merchant portal. If you are having issues, you can always reach out to storefronthelp@doordash.com.

The plugin utilizes the Storefront SDK. It is a simple javascript library that runs in the browser, and depending on the configuration provided injects a floating button in the page. The button and the link generated is conversion tracked. It tracks when the user sees the button, clicks on it and finally when they complete their order on Storefront.
The SDK does not execute any PHP code on your server side. It can be simply thought of like any other third party Javascript libraries e.g. Google Tag Manager, Maps etc.

 [Storefront Terms of Service and Privacy Policy](https://help.doordash.com/merchants/s/storefront-services-terms-of-service?language=en_US) 



== Installation ==
1. Navigate to the "Plugins" section in the dashboard.
2. Click "Add New"
3. Search for the "DoorDash Storefront" plugin
4. Click "Install Now"
5. Activate the plugin on the plugin dashboard
6. Click on settings link inside the "Storefront Online Ordering by DoorDash" plugin row.
7. Set all required fields like "Anchor Store ID", "Business ID" etc.
8. Go to "Appearance" > "Widgets", select "Storefront by DoorDash" and drag to a Storefront sidebar.



== Frequently asked questions ==
= What setup is required to use this plugin =
You should have completed onboarding with Doordash - Storefront, in case you are unsure, check with storefronthelp@doordash.com.
= What is BusinessId, where can I get this = 
Log in to your merchant portal business account and copy the number at the end of your URL. For example, if your url is "https://www.doordash.com/merchant/summary?business_id=123456", then your business ID is "123456" 
= I don’t want to use this plugin, what are my other options =
You can add the Storefront link manually in your site, Doordash Merchant Portal has instructions for getting the link to your Storefront Site.
